🌴 Bastar Explorer – Tourist Booking Website

A modern and responsive tourist website built using HTML, CSS, and JavaScript that helps users explore popular places in Bastar and plan their trips بسهولة.



🚀 Features
🏞️ Explore popular tourist places (Chitrakoot, Tirathgarh, Kanger Valley)
📖 View place details on click
🚗 Book trips easily
💰 Automatic cost calculation based on number of persons
📩 Contact form for user queries
📱 Fully responsive design
🎨 Modern UI with glassmorphism effects



🛠️ Tech Stack
HTML – Structure
CSS – Styling & Responsive Design
JavaScript – Functionality & Logic


📂 Project Structure
Bastar-Explorer/
│
├── index.html
├── style.css
├── script.js
└── images/


⚙️ How to Run
Download or clone the repository
Make sure all files are in the same folder
Open index.html in your browser


🎯 How It Works
Click on any place card to view details
Click "Book Now" to select a destination
Enter number of persons
Click "Calculate Cost" to get total price
Confirm booking with "Confirm Booking"
